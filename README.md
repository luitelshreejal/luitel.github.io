# Portfolio

React + Vite + TypeScript. Deploy: GitHub Actions → Pages (`dist/`). `vite.config.ts` `base` must match the repo name for project Pages (`/luitelshreejal/`).

https://luitelshreejal.github.io/luitelshreejal/

```bash
npm install
npm run dev    # http://localhost:8080/luitelshreejal/
npm run build
npm run preview
npm run lint
npm test
```
